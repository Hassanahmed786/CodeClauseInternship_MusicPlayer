# Python Music Player

## Description
- Simple, lightweight music player built with Python.
- Easy-to-use GUI for managing music playback.

## Features
- **Play**: Play music files.
- **Pause**: Pause the current track.
- **Stop**: Stop playback.
- **Load**: Select and play music files.
